# ChatCraft Admin Project

## Project Overview

This is the admin application for ChatCraft, providing global administration capabilities for the entire ChatCraft platform. From this application, administrators can:

- View and manage all tenants
- Monitor messages across all subscribers
- Track usage and utilization metrics
- Manage email communications
- Oversee subscriber activity and analytics
